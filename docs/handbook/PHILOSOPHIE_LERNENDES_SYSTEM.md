# Philosophie Lernendes System

## Leitidee

EduCode Lab ist ein lernendes Entwicklungssystem.

Grundmuster:

1. realer Bedarf
2. verstandenes Problem
3. saubere Einzelloesung
4. universelles Modul
5. ueberpruefte Verbesserung

Leitsatz:

Aus Ideen werden Bausteine, aus Erfahrungen wird Wissen, aus Wissen waechst ein System, das mit seinen Menschen lernt.

## Produktprinzipien

1. Beduerfnisse vor Funktionen
2. Unterstuetzung statt Entmuendigung
3. Nachvollziehbarkeit vor Blackbox-Automatisierung
4. Wiederverwendbarkeit vor Einmal-Loesung
5. Lernwirkung vor Feature-Effekt

## EduCode Lernkreislauf

1. Beobachten
2. Bedarf verstehen
3. Idee strukturieren
4. Loesung entwerfen
5. Erproben
6. Wirkung messen
7. Erkenntnis sichern
8. System verbessern

Jede neue Idee und jede implementierte Funktion wird ueber diesen Kreislauf gefuehrt.

## KI-Leitplanken

Fuer Lernende:

1. keine unkontrollierte Komplettloesung
2. gestufte Hilfen und Diagnose von Denkfehlern
3. Foerderung von Verstehen und Selbststaendigkeit

Fuer Lehrkraefte:

1. Entwuerfe und Varianten statt Ersatz didaktischer Entscheidungen
2. Reduktion von Routineaufwand
3. dokumentierte, begruendbare Vorschlaege
