# Core Principles

1. Needs first: Jede Entwicklung startet mit einem realen Bedarf.
2. Human in command: KI unterstuetzt Entscheidungen, ersetzt sie nicht.
3. Reusable by default: Einzelfaelle auf Wiederverwendbarkeit pruefen.
4. Evidence over opinion: Wirkung ueber Beobachtung und Feedback belegen.
5. Learn in loops: Jede Umsetzung erzeugt neues Systemwissen.

## Prueffragen vor Umsetzung

1. Welches konkrete Problem wird geloest?
2. Wen betrifft es und wie haeufig tritt es auf?
3. Welche einfachere Alternative gibt es?
4. Welche Risiken entstehen (Datenschutz, Fehlverhalten, Komplexitaet)?
5. Was ist ein messbares Erfolgskriterium?
