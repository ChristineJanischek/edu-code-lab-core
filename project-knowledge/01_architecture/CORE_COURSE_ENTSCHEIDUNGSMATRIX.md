# Core Course Entscheidungsmatrix

## Ziel

Einheitlich entscheiden, wo eine neue Funktion umgesetzt wird.

## Entscheidungsfragen

1. Ist die Funktion fachunabhaengig und in mehreren Kursen nutzbar?
- Ja: Core-Kandidat.

2. Enthält die Funktion primär fachliche Regeln, Quellen oder didaktische Varianten?
- Ja: Course-Konfiguration, ggf. mit Core-Technik.

3. Betrifft die Funktion nur eine einzelne Aufgabe oder Unterrichtseinheit?
- Ja: Aufgabe oder Kurs.

4. Wird dieselbe Logik bereits mehrfach in verschiedenen Kursen dupliziert?
- Ja: Extraktion in Core vorschlagen.

5. Erzeugt die Funktion neue Integrations- oder Vertragsanforderungen?
- Ja: Integrationsvertrag und Migrationshinweise aktualisieren.
