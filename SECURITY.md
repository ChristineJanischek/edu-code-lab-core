# Security Policy

## Meldung von Sicherheitsluecken

Bitte keine oeffentlichen Issues fuer kritische Luecken.
Sicherheitsrelevante Meldungen direkt an die Maintainerin.

## Mindestschutz

- Branch Protection auf `main`
- Pflicht-Reviews fuer PRs
- Security-/Dependency-Scan in CI
- Secret-Leaks vermeiden (keine Tokens/Schluessel committen)
